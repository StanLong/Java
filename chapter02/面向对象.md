# 面向对象

## this

```java
/*
 this  不能用在静态方法中, 只有成员方法才能用
 
*/

public class Test{
    
    // 成员变量
    String str;
    
    public static void main(String[] args){
        // 直接编译就报错了 
        // Test.java:29: 错误: 无法从静态上下文中引用非静态 变量 this
        Person.m1(); 
        
        // System.out.println(str); //也会报错， 静态方法不能访问成员方法
    }
}

class Person{
    
    // 成员变量
    String name;
    
    Person(){
        
    }
    
    Person(String name){
        this.name = name;
    }
    
    // 成员变量必须使用引用.的方式访问，而用引用则必须要有对象。
    // 静态方法的调用是不需要有对象的，使用类名.方法名的方式调用. 所以静态方法中不能使用this
    public static void m1(){
        // System.out.println(name); 
        // System.out.println(this.name); 
        
        // 如果想要访问name只能
        Person p1 = new Person("StanLong");
        System.out.println(p1.name);
    }
}
```
